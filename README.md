# Netflix Clone

A responsive Netflix landing page clone built using HTML5 and CSS3. This project is a front-end recreation of the official Netflix homepage, designed to practice modern web development concepts such as responsive layouts, Flexbox, positioning, gradients, and UI design.

The goal of this project was to improve my understanding of HTML and CSS by replicating a real-world website as closely as possible while maintaining clean and organized code.

## Live Website

🔗 Website: https://shoumflix-2007.netlify.app/

## Features

* Responsive layout for different screen sizes
* Netflix-inspired landing page design
* Navigation bar with language selector and sign-in button
* Hero section with call-to-action
* Trending movies section
* "More Reasons to Join" cards
* Modern typography and gradients
* Clean folder structure
* Organized and readable code

## Technologies Used

* HTML5
* CSS3

## Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/Netflix-Clone.git
```

Navigate to the project folder:

```bash
cd Netflix-Clone
```

Open `index.html` in your preferred web browser.

No additional dependencies or installations are required.
